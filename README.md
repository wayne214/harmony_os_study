## HarmonyOS开发学习实践

### 一个比较完整的开眼APP
主要用于学习了解HarmonyOS，基础组件的使用，页面布局，页面展示，页面跳转，网络请求等基础内筒。

#### 项目整体结构
- 首页
- 发现
    - 关注
    - 分类
    - 主题
  - 热门
    - 日排行
    - 月排行
    - 总排行
- 我的
- 内容详情页

#### 实现功能
- 列表的展示，上拉加载，下拉刷新
- 列表卡片布局展示
- 详情页上部分视频播放
- 详情页下部分数据列表展示
- Tab列表联动展示
- 层叠布局

#### 封装工具类
- 网络请求：HttpUtils
- 日志类
- 用户首选项存储工具类：StorageUtils
- 吐司提示：ToastUtils
- 窗口宽高：WindowUtils

#### 封装组件
- Dialog弹框
- 顶部标题
- 视频组件
